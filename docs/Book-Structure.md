# 📄 Book Template Structure

The book template uses the standard LaTeX `book` class, which is structured into `\frontmatter`, `\mainmatter`, and `\backmatter`.

## 🗂️ File Organization

*   `main.tex`: Entry point containing structure setup and chapter imports.
*   `hcmus-report-template-book.sty`: Custom package handling page layout, math formatting, and list styles.
*   `img/`: Holds graphic assets (such as university logos and figures).
*   `ref/ref.bib`: BibTeX file containing citation references.
*   `appendix/appendix.tex`: Appendices.

---

## 📝 Document Chapters (`content/`)

*   `content/title.tex`: The book cover page format.
*   `content/guide.tex`: A detailed, pre-written user guide (can be removed from `main.tex` once read).
*   `content/introduction.tex`: Chapter 1 (Introduction).
*   `content/related-works.tex`: Chapter 2 (Related Works).
*   `content/proposed-method.tex`: Chapter 3 (Proposed Method).
*   `content/experiments.tex`: Chapter 4 (Experiments & Discussion).
*   `content/conclusion.tex`: Chapter 5 (Conclusion).
